How to Use the Scoresheet

1. Select Teams
Choose the Home Team and Away Team from the dropdown menus. Rosters load automatically.

2. Assign Lag Winner / Lag Loser
Tap a player name from either roster. The system fills Lag Winner and Lag Loser automatically. This can be changed if the wrong player is chosen.

3. Enter Total Match Innings
Use the + / – buttons to adjust the total innings.

4. Enter Player Stats
Each player has the following stats:
- HC = Handicap
- Wins = Games won
- Safeties = Defensive shots
- 8B = 8 on Break(no scratch)
- B+R = Break and Run
- E8 = ANY 8 Ball error resulting in loss of game.

5. Match Winner
This field auto‑fills based on total wins.

6. Submit Match
Press “Submit Match”. The button disables and a message appears:
“Please wait while your results are submitted…”
When complete:
“Match submitted successfully. You may now enter the next match.”
