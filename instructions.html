How to Use the Scoresheet

1. Select Teams
Choose the Home Team and Away Team from the dropdown menus. Rosters load automatically.

2. Assign Lag Winner / Lag Loser


1st name tapped will be set as Lag Winner, 2nd player tapped will be set as Lag Loser. This can be changed if the wrong player is chosen by tapping names again.

3. Enter Total Match Innings


Hash is determined by TOTAL Innings - Safeties / Wins thus there is NO need for a game by game inning count
Use the + / – buttons to adjust the total inning count.

4. Enter Player Stats


Each player has the following stats:
- HC = Handicap - These are auto filled but adjustable in case of shooting a new player or clerical error
- Wins = Games won
- Safeties = Defensive shots
- 8B = 8 on Break(no scratch)
- B+R = Break and Run
- E8 = ANY 8 Ball error resulting in loss of game. - Coach Check Box = Check the box when a player takes a coach. Resets when the game is over.

5. Match Winner


This field auto‑fills based on total wins.

6. Transfer


Creates a scannable QR code for scoresheet sharing. Useful if the scorekeeper needs to leave.

7. Notes


Clicking opens a box to leave the LO notes. Click again to close the box.

8. Timer


Clicking opens a pop up timer

9. Submit Match


Invisible until there is a match winner. Press “Submit Match”. The button disables and a message appears:
“Please wait while your results are submitted…”
When complete: Match submitted successfully. You may now enter the next match.”
