# ROWN Pool League – Digital Scoresheet

This repository contains the official digital scoresheet used for weekly league match reporting.
The scoresheet is mobile‑friendly and designed for fast, accurate stat entry.

------------------------------------------------------------
LIVE SCORESHEET
------------------------------------------------------------
https://rownpoolleague.github.io/scores/

------------------------------------------------------------
HOW TO USE THE SCORESHEET
------------------------------------------------------------

1. SELECT TEAMS
Choose the Home Team and Away Team from the dropdown menus.
Rosters load automatically based on your selection.

2. ASSIGN LAG WINNER / LAG LOSER
Tap a player name from either roster.
The system fills:
- Lag Winner
- Lag Loser
If you tap the wrong player you can still change it.

3. ENTER TOTAL MATCH INNINGS
Use the + / – buttons to adjust the total innings for the match.
There is no need to track innings per games as hash is Determined by
TOTAL Innings - Safeties / Wins

5. ENTER PLAYER STATS
Each player has the following tracked stats:
- HC = Handicap
- Wins = Games won
- Safeties = Defensive shots
- 8B = 8 on Break
- B+R = Break and Run
- E8 = ANY error on the 8 resulting in a loss.

All stat fields use +/– buttons for fast mobile entry.

5. MATCH WINNER
This field auto‑fills based on total wins.

6. SUBMIT MATCH
Invisible until there is a match winner. Press "Submit Match".

You will see:
"Please wait while your results are submitted..."

The button disables to prevent double submissions.

When complete:
"Match submitted successfully. You may now enter the next match."
